== Description ==
This plugin is used to integrate KwikPaisa payment gateway with Woocommerce.

Pay Your Order with KwikPaisa is the simplest of the available ways to do payment integration to your website. 

The person who clicks the Pay via KwikPaisa button will be taken to Pay with KwikPaisa website where user will complete the transaction via KwikPaisa and Pay with KwikPaisa returns back the browser handle to the website after payment confirmation. Website will also receive a notification, which will be a server-to-server call as soon as the payment is done.

== Installation ==
Installation

Login your Wordpress admin and navigate to Plugins->Add New 

Find the WooCommerce KwikPaisa Pay plugin and activate the module.

After the successful activation of module, Visit the WooCommerce => settings page, and click on the Checkout tab.

Click On Checkout and display "KwikPaisa NEO Bank" payment gateway.

Enable the Payment Method, and Set all details and  Click Save.(See screenshot)

Now you can see *Process to KwikPaisa * payment option on checkout page.

For Create Merchant ID Go to <a href="https://dashboard.kwikpaisa.com" target="_blank">KwikPaisa Merchant Dashboard</a>.

* Signup/Sign In on https://kwikpaisa.com and fill in the profile details and bank details.
